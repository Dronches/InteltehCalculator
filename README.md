# Калькулятор Интелтех
Калькулятор, реализующий:
* Часть стандартных функций калькулятора Windows;
* Интерфейс калькулятора Windows;
* Многопоточность;
* Две очереди и отображение их количества;
* Имитация длительности выполнения
* Сохранение настроек положения приложения;
* Хранение данных в течение сеанса.
